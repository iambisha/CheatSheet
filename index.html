Basic tag

< html >

it is the start and end of the website.

< head >

it is the metaphor of the website it contain meta data.

< title >

it helps to give title to the website.it goes inside head tag.

< body >

all visible things of our website goes inside of body tag.

< h1 > - < h6 >

they are the headings, sizes decreases simultaneously from h1 to h6.

< p >

all paragraphs of the page goes inside this tag.

< strong >

it makes the text bold.

< em >

it emphasizes or italisize the text.

< ol >

this tag helps to put items in order list

< ul >

this tag helps to put items in un-order list

< li >

all the list items of ul or ol tag goes here

< br >

it is a self closing tag which helps to add break
